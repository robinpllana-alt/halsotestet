# Hälsotestet

Hälsotestet är en svensk plattform för vetenskapligt baserade självskattningstester inom psykisk och fysisk hälsa.

## Vision

Att skapa Sveriges mest professionella plattform för digitala självskattningstester.

## Mål

- Modern och snabb webbplats
- Mobilanpassad
- Vetenskapligt baserade tester
- Enkelt att lägga till nya tester
- Professionell adminpanel
- Skalbar plattform
